# Fiesta

## Quick start

1. Скачайте или клонируйте проект.
2. Откройте `index.html` в браузере либо поднимите локальный сервер, чтобы увидеть страницу.
3. Если нужны правки, редактируйте `index.html`, `styles.css` и `script.js` — дополнительных сборок нет.

### Если предпросмотр пишет NotFound

Некоторые редакторы и облачные среды показывают 404/NotFound, если страница не обслуживается локальным сервером. Чтобы увидеть предпросмотр прямо здесь:

1. Запустите сервер из корня репозитория: `./preview.sh` (по умолчанию порт 8000; можно задать другой через переменную `PORT`).
2. Откройте `http://localhost:8000/` — главная страница загрузится, а ссылки на PDF и `menu.html` будут работать.
3. В VS Code/Codespaces выберите «Open in Browser/Preview» на том же порту, чтобы встроенный предпросмотр перестал показывать NotFound.
